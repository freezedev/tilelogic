tilemap
=======

Tilemap for JavaScript games

```javascript
var tilemap = new Tilemap(10, 10);
// creates a 10 by 10 tilemap
```

Helper functions
```javascript
tilemap.each(function() {

});
```

```javascript
tilemap.map(function() {

});
```

Compare two tilemaps with each other
```javascript
tilemap.equals(new TileMap(10, 10));
```
